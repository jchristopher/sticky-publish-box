== Description ==
Set the side publishing box on the WP post editor to stay on the side. Helpful for when you a lot of custom fields.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the 'sticky-publish-box' folder to the `/wp-content/plugins/` directory or install via the WP admin panel
1. Activate the plugin through the 'Plugins' menu in WordPress
